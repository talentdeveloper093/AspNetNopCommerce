HOW TO INSTALL A NOPCOMMERCE THEME

  • When you download a theme from Global technologic it comes in Zip Filder i.e (Nop.zip)
  • You need to extract the Nop.zip file
  • There are Themes folders.
  • Select Theme floder & copy the theme folder.
  • Go to theme folder of your nopCommerce installation.
  • Select Themes folder & Paste( Presentation\Nop.Web\Themes)
  • Select plugin & Paste( Presentation\Nop.Web\Plugins)

THEME SETTING

  • 1. Now login in your web site and go in the Administration to install the Theme
  • 2. Go to Administration --- Configuration --- Settings --- General Settings on clicking --- Select Everything theme--- Save
